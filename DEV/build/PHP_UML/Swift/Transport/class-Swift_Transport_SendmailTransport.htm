Swift_Transport_SendmailTransport

Class Swift\Transport\Swift_Transport_SendmailTransport extends Swift\Transport\Swift_Transport_AbstractSmtpTransport

  • SendmailTransport for sending mail through a sendmail/postfix (etc..) binary.

    Supported modes are -bs and -t, with any additional flags desired.
    It is advised to use -bs mode since error reporting with -t mode is not possible.
