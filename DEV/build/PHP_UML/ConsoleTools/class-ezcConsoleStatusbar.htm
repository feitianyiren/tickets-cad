ezcConsoleStatusbar

Class ConsoleTools\ezcConsoleStatusbar

  • Creating and maintaining status-bars to be printed to the console.

    <code> // Construction $status = new ezcConsoleStatusbar( new ezcConsoleOutput() );
    // Set option $status->options['successChar'] = '*';
    // Run statusbar foreach ( $files as $file ) { $res = $file->upload(); // Add status if form of bool true/false to statusbar.
    $status->add( $res ); // $res is true or false }
    // Retreive and display final statusbar results $msg = $status->getSuccess() . ' succeeded, ' . $status->getFailure() . ' failed.'; $out->outputText( "Finished uploading files: $msg\n" ); </code>
