ezcConsoleTableRow

Class ConsoleTools\ezcConsoleTableRow implements Countable, Iterator, ArrayAccess

  • Structure representing a table row in ezcConsoleTable.
    This class represents a row in a table object. You can access the properties of the row directly, but also access the cells of the row directly, as if the object was an array (index 0..).

    <code> // Create new table row $row = new ezcConsoleTableRow();
    // Set default format of the row's cells $row->format = 'headline';
    // On the fly create the cell no 0 $row[0]->content = 'Name'; // On the fly create the cell no 1 $row[1]->content = 'Cellphone';
    // Change a setting on cell 0 $row[0]->align = ezcConsoleTable::ALIGN_CENTER;
    // Iterate through the row's cells.
    foreach ( $row as $cell ) { var_dump( $cell ); }
    // Set the default align property for all cells in the row $row->align = ezcConsoleTable::ALIGN_CENTER;
    </code>
    This class stores the rows for the {@link ezcConsoleTable} class.
