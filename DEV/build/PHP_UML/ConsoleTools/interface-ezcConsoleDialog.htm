ezcConsoleDialog

Interface ConsoleTools\ezcConsoleDialog

  • Interface that every console dialog class must implement.

    Console dialogs can either be used on their own or using the {@link ezcConsoleDialogViewer} (recommended). In the dialog viewer, a dialog is instanciated and displayed in a loop, until it receives a valid result value.
