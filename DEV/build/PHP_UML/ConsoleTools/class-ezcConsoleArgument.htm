ezcConsoleArgument

Class ConsoleTools\ezcConsoleArgument

  • The ezcConsoleArgument class represents an argument on the console.
    This class is the container to store information about an argument submitted to a shell application. It is used to define the appearance of an argument before parsing the parameter string and contains the received value afterwards. Argument objects are stored in an instance of the set class {@link ezcConsoleArguments} which is stored in {ezcConsoleInput::$argumentDefinition}.
