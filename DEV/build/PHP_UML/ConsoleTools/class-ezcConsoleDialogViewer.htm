ezcConsoleDialogViewer

Class ConsoleTools\ezcConsoleDialogViewer

  • Utility class for ezcConsoleDialog implementations.
    This class contains utility methods for working with {@link ezcConsoleDialog} implementations.

    To display a dialog in a loop until a valid result is received do:
    <code> // Instatiate dialog in $dialog ...
    ezcConsoleDialogViewer::displayDialog( $dialog ); </code>
    For implementing a custom dialog, the method {@link readLine()} method can be used to read a line of input from the user.
