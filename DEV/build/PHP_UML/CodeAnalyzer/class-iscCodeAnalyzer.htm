iscCodeAnalyzer

Class CodeAnalyzer\iscCodeAnalyzer

  • searchs through source tree and collects infos about found classes and files
    Some basic statistics are collected:
    - LoC - count of elements (classes, methods, ...) - Missing DocTags per element - used DocTags
