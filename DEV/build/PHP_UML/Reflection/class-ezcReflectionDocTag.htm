ezcReflectionDocTag

Class Reflection\ezcReflectionDocTag

  • Represents a doc tag in the php source code comment.

    This class is used as standard implementation for representing annotations. It is only used if no specialized tag class could be found deriving from this class.

    The comment line is tokenized by at spaces and if no further structure is recognized, tokens are available at getParams.
